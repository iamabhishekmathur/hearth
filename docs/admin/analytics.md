# Analytics

Applies to: Hearth Cloud and self-hosted Hearth.

Analytics help admins understand adoption, usage, and operational patterns across the workspace.

[[toc]]

## What Analytics Show

Analytics can include:

- Active users.
- Chat usage.
- Task activity.
- Routine usage.
- Integration usage.
- Token or provider consumption where available.
- Feature adoption.
- Trends over a selected time range.

## How to Use Analytics

Use analytics to answer:

- Which teams are adopting Hearth?
- Which workflows repeat enough to become routines?
- Which integrations create the most value?
- Are tasks reaching review and done?
- Are routines healthy or failing?
- Do governance or compliance settings need adjustment?
