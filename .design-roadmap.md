# Design-Driven Product Roadmap (Hidden)

Improvements identified from the new Hearth design system that go beyond pure UI reskin.
These represent product enhancements suggested by the design that are not yet implemented.

## Chat Enhancements
- **Display title for active chat** — Large Fraunces display title (42px) showing the session name, with metadata (started time, shared team, tool call count) as subtitle
- **Session pill tabs** — Replace tab bar with horizontal pill-style tab row showing active session + recent sessions inline
- **Right sidebar context rail** — Contextual panel showing: connected data sources, related past chats, team activity feed, and proactive suggestions (e.g. "Make this a routine?")
- **Composer chips** — Rich composer bar with `@ memory`, `/ skills`, `attach`, model selector chips, and `⌘↵` shortcut hint
- **Proactive routine suggestion** — Hero card in sidebar suggesting scheduling recurring chats based on frequency detection
- **Inline data viz** — Horizontal bar charts and sparklines rendered inline in assistant messages (HBars, HSpark components)

## Workspace Enhancements
- **Board view with cards** — 3-column grid of typed cards (chat, artifact, doc) with kind icons, tags, pin state
- **Breadcrumb navigation** — Finance > Q3 close breadcrumb showing workspace hierarchy
- **Filter pills** — Type filtering (All/Chats/Artifacts/Docs) with count badges

## Routines Enhancements
- **Routine step visualization** — Visual step-by-step pipeline view showing numbered steps with tool labels and connector lines
- **Run history panel** — Recent runs with status dots (ok/warn/err) and summary text
- **Grid layout for routine list** — Rich row layout: icon + title/cadence + next run + status dot + owner

## Skills Enhancements
- **Skill detail panel** — Right panel showing full skill detail: usage sparkline, input schema with types, tool inventory, installed-by avatars
- **Skill IO display** — Typed input/output specification (icon + label + type + required badge)
- **Usage sparkline** — Inline sparkline chart showing skill usage over time

## Memory Enhancements
- **Collection sidebar** — Left nav with team/personal/auto-learned collections with entry counts
- **Memory type badges** — Color-coded type labels (fact=blue, rule=purple, preference=amber, contact=green)
- **Auto-learned hero card** — Proactive card: "3 new things learned this week — approve to add to team memory"

## Activity Enhancements
- **Timeline with day headers** — Activity feed with date dividers and time-stamped entries
- **Typed activity icons** — Color-coded icons per activity type (chat, routine, skill, share, anomaly, member)
- **Weekly stats panel** — This-week summary: chats, routine runs, anomalies, hours saved
- **Most active skills chart** — Horizontal bar chart of top skills by usage

## Admin Enhancements
- **Stats row** — 4-column summary: members, monthly active, tokens, plan
- **Members table** — Rich member table with avatar, role pill, last active, status badges
- **Token usage sparkline** — Monthly token consumption with trend sparkline
- **Billing panel** — Plan details, next invoice, add seats, change plan

## Onboarding
- **Split-screen onboarding** — Left: brand + step progress, Right: active step content
- **Guided option cards** — Pre-built starting prompts as selectable hero cards with recommended badge
- **Data source ready badge** — Connected source indicator with "ready" badge

## Settings
- **Settings nav sidebar** — Icon + label nav: Account, Workspace, Integrations, Notifications, Models, Privacy, Billing, Export
- **Model radio selector** — Radio-style model picker with name, description, and "recommended" badge
- **Toggle rows** — Setting toggles with label + description + switch
- **Danger zone** — Red-bordered card for destructive actions (transfer/delete workspace)

## Design System
- **Dark mode parity** — Full dark theme via `data-theme="dark"` attribute
- **Fraunces display font** — Editorial serif for titles and hero numbers
- **Voice guidelines** — "Crisp + professional. Linear / Stripe style." No exclamation marks, no emoji in product UI
